--- layout: post title: Neutron IDE V11.10 Released categories: - release ---

Today version 11.10 of Neutron IDE is released. With this version, Neutron has changed to time based releases. This will help hopefully make the version number actually mean something. Here is what is included in Version 11.10.

  • Ability to edit images with Pixlr
  • Upgrade to Ace Editor v0.2.
  • Several bug fixes and UI enhancements.

Download the latest release at Git Hub (recommended) or download the tarball.

For installation and upgrade directions see the wiki.

Update: 10/09/2011 @ 11:34AM CST
Pixlr has stopped working for me because it is not loading images from URL. It is a general issue with Pixlr and not specific to Neutron IDE. Hopefully they will fix it soon.

Update: 10/10/2011 @ 10:37AM CST
Looks like Pixlr is working again. :-)
