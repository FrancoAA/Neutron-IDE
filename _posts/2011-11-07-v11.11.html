Today, version 11.11 of Neutron IDE is released. This month's release packs a lot of changes. Here is what is included in Version 11.11.

  • Save Session: Option to save current files your working on. If enabled then the same files will be reopened when you return.
  • Save All: Save all open files.
  • Close All: Close all open files.
  • Added easy access to the editor settings via the toolbar menu. Also updates settings without a page refresh.
  • Switched out Pixlr photo editor for Phoenix.
  • Added help menu for keyboard shortcuts.
  • Integrated Kendo UI to improve UI consistency and overall look.
  • Added Google Analytics to track IDE use.

Download the latest release at Git Hub (recommended) or download the tarball.

For installation and upgrade directions see the wiki.
