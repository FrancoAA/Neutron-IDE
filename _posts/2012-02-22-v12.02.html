--- layout: post title: Neutron IDE V12.02 Released categories: - release ---

Today, I'm releasing Version 12.02 of Neutron IDE. The terminal is now very stable and is great way to get command line access to your server in any modern web browser. Listed below is a list of everything included in this release.

  • Terminal Fixes and Upgrades: Several bugs were fixed and features such as the ability to resize the terminal, the integration of dtach to support persistent terminals even if you lose connectivity, editable terminal preferences, and scroll buttons were added.
  • Upgrades to executable: The executable neutronide.py was upgraded to include default backgrounding of the proccess, default output to log files, and a change of default log and database file locations. If upgrading please see the installation page on the wiki for more details. These upgrades make neutronide.py easier to start and run automatically on boot.
  • Ace Editor Updated: Ace editor was updated to the lastest version which includes features like code folding and support for additional languages.

Download the latest release at Git Hub (recommended) or download the tarball.

For installation and upgrade directions see the wiki.
